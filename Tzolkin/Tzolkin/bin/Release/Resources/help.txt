|-- Знаки ЦОЛЬКИН ( По книге Георгия Лазарева « Тайна священного календаря майя» ):


ИМИШ ( Дракон , Мировое Дерево, Крокодил) — День благоприятен для духовной работы, начала нового дела. В этот день приходят Божественные откровения, и открываются тайны природы.


ИК ( Ветер ) - День особенно хорош для духовной работы,развития силы разума , интеллекта , памяти. Благоприятен для общения.


АКБАЛЬ ( Ночь ) - Магический день. День трансформации и перемен, дает новые возможности, успех в делах и личной жизни, привносит в жизнь стабильность. Благоприятен для всех видов деятельности.


КАН ( Зерно, Игуана, Ящерица ) - Хороший день для новых начинаний, установления контактов с мирами, развязывания кармических узлов,устранения негативных энергий. Магический день творческой силы, плодородия,хорош для посева, зачатия. Он связан с духовным и материальным ростом.


ЧИКЧАН ( Змей ) - День магии и тайны, день общения с духами, и проведения ритуалов, связанных с дождем. Удачный день, благоприятный для торговцев. Дает человеку мощную энергию, сексуальную ауру, открывает вход в магические миры. В этот день люди становятся сильны и физически, и ментально. Хороший день для возвращения потерянного и забытого, для любовного союза.


КИМИ ( Смерть , Череп ) - В день смерти хорошо вспоминать предков..Благоприятно находить доступ к более высокому знанию, заканчивать невыгодные отношения и сделки и не начинать новые дела.


МАНИК ( Олень, Рука ) -День любви, силы,энергии,магии. День Леса. В этот день вам обязательно помогут,протянут руку помощи. Это день встречи с важными людьми, успехов в делах.


ЛАМАТ ( Звезда , Кролик )- Путеводный, судьбоносный день, благоприятный для всех начинаний. День благодарения, полный радости и приятных сюрпризов. Магический день. В житейском плане Звезда указывает на золото, серебро, удачную коммерческую сделку.


МУЛУК ( Вода , Луна ) - День Луны это связь с более высокой жизнью, природой и космосом, баланс правосудия и любви, духовный путь. Благоприятно проливать свет на тайные вещи,понимать закон причины и следствия, прощать и быть прощенными.


ОК ( Собака ) - Магический день поминовения. В этот день открывается щель между мирами. Собака, встреченная в этот день- это Нагуаль,поэтому наблюдайте за ней , и ее поведением. Это хороший знак.


ЧУЭН ( Мастер , Обезьяна ) - Хороший день, чтобы «начать новую жизнь» в любой сфере. День творчества во всех его проявлениях. День магии, церемоний, инициации.


ЭБ ( Дорога , Путь ,Человек) - Это лучший день для бизнеса,общественного труда,работы с документами, ценными бумагами. Очень благоприятен для путешествия.


БЕН (Маис , Тростник ) - Мужской, активный день, благоприятный для любых контактов,действий,особенно для шаманской и целительной работы. Это — день защиты жизни, детей, и беременных женщин. Нагуаль БЕН имеет дело с аспектами, касающимися семей, групп людей, коллективов и общин.


ИШ ( Ягуар , Волшебник ) - День священной Земли, дающий бонусы, связанные с землей, собственностью. День благоприятствует изменению всех отрицательных аспектов жизни, улучшению личных дел, поправлению здоровья и материального положения. Это хороший период для женщин, особенно во время путешествий, знакомств, свадеб.


МЕН ( Орел ) - Священный день. Благоприятен для взаимодействия в любой сфере. Лучший день для любви, радости, веселья. Хороший день для общественных событий.


КИБ ( Сова, Гриф ) - День очищения, исцеления. Благоприятен для лечения, посещения священных мест, и проведения ритуалов поминовения предков.


КАБАН ( Земля , Благовоние ) - День благоприятен для целенаправленной деятельности как в духовном, так и в материальном плане. Это день творческого созидания. Движения и Энергии. Благоприятен для установления контактов, согласования отношений, решения споров.


ЭЦНАБ ( Зеркало , Кремень ) - День несет магические и жизненные испытания. Может нести боль и страдания, а может отсекать от человека болезни и дурные помыслы. В этот день предстоит сделать выбор, порой нелегкий.


КАВАК ( Молния , Гроза ) - День хорош для зачатия детей, целительства , начинания новых дел в бизнесе, привлечения богатства и изобилия.


АХАУ ( Солнце , Владыка ) - День благоприятен для творчества, семейных традиций, а так же для прорицания. День дает энергию и силу для преодоления любых препятствий, излечения любых недугов, достижения выбранной цели.
